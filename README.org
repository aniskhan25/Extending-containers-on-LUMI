* Guide
This guide shows how to extend the current LUMI AI Factory container with Poppler.
The current LUMI AI Factory multi-framework container is available at =/appl/local/laifs/containers/lumi-multitorch-latest.sif=.

Building containers requires memory, so run these commands on a LUMI login node and allocate resources before extending the container:

#+BEGIN_SRC bash
export PROJECT=<project>
#+END_SRC

#+BEGIN_SRC bash
export SIF=/scratch/$PROJECT/$USER/lumi-poppler.sif
#+END_SRC

#+BEGIN_SRC bash
srun --account=$PROJECT --partition=small --ntasks=1 --cpus-per-task=8 --mem=120G --time=02:00:00 --nodes=1 --pty bash
#+END_SRC

#+BEGIN_SRC bash
module purge
module load CrayEnv
module load PRoot

mkdir -p /tmp/$USER
export SINGULARITY_TMPDIR=/tmp/$USER
export SINGULARITY_CACHEDIR=/tmp/$USER
#+END_SRC

Create a definition file (e.g., =poppler-lumi.def=) where you define the packages to be installed:

#+BEGIN_SRC bash
Bootstrap: localimage
From: /appl/local/laifs/containers/lumi-multitorch-latest.sif

%post
    apt-get update
    DEBIAN_FRONTEND=noninteractive apt-get install -y poppler-utils
    apt-get clean
    rm -rf /var/lib/apt/lists/*
#+END_SRC

Use Singularity to build the new container:

#+BEGIN_SRC bash
singularity build $SIF poppler-lumi.def
#+END_SRC

From the allocated shell:

#+BEGIN_SRC bash
module use /appl/local/laifs/modules
module load lumi-aif-singularity-bindings
singularity exec $SIF pdftoppm -png sample.pdf sample
#+END_SRC
