* Introduction
This guide shows how to extend the current LUMI AI Factory container with Poppler and Python packages.

* Extend the LUMI AI Factory Container
The current LUMI AI Factory multi-framework container is available at =/appl/local/laifs/containers/lumi-multitorch-latest.sif=.

** Allocate resources
Building containers requires memory, so it is recommended to allocate resources before extending the container:

#+BEGIN_SRC bash
export PROJECT=<project>
#+END_SRC

#+BEGIN_SRC bash
export SIF=/scratch/$PROJECT/$USER/lumi-poppler.sif
#+END_SRC

#+BEGIN_SRC bash
srun --account=$PROJECT --partition=dev-g --ntasks=1 --cpus-per-task=7 --mem=120G --time=02:30:00 --nodes=1 --pty bash
#+END_SRC

** Load necessary modules
#+BEGIN_SRC bash
module purge
module load CrayEnv
module load PRoot

mkdir -p /tmp/$USER
export SINGULARITY_TMPDIR=/tmp/$USER
export SINGULARITY_CACHEDIR=/tmp/$USER
#+END_SRC

** Create a definition file
Create a definition file (e.g., =poppler-lumi.def=) where you define the packages to be installed:

#+BEGIN_SRC bash
Bootstrap: localimage
From: /appl/local/laifs/containers/lumi-multitorch-latest.sif

%post
    apt-get update
    DEBIAN_FRONTEND=noninteractive apt-get install -y poppler-utils
    apt-get clean
    rm -rf /var/lib/apt/lists/*
#+END_SRC

** Build the container
Use Singularity to build the new container:

#+BEGIN_SRC bash
singularity build $SIF poppler-lumi.def
#+END_SRC

** Install Python packages
Create a virtual environment inside the container for Python packages:

#+BEGIN_SRC bash
module use /appl/local/laifs/modules
module load lumi-aif-singularity-bindings
singularity shell $SIF
#+END_SRC

Inside the container:

#+BEGIN_SRC bash
python -m venv h5-env --system-site-packages
source h5-env/bin/activate
pip install pdf2image h5py
#+END_SRC

** Run a script

#+BEGIN_SRC bash
module use /appl/local/laifs/modules
module load lumi-aif-singularity-bindings
srun singularity exec $SIF bash -c 'source h5-env/bin/activate && python my-script.py'
#+END_SRC

** Test Poppler
#+BEGIN_SRC python
  from pdf2image import convert_from_path
  images = convert_from_path('sample.pdf', fmt='png')
#+END_SRC
